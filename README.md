# AdvPL & TLPP Tutor

Bem-vindo ao tutor interativo de AdvPL e TLPP (TOTVS Language Plus Plus). Este projeto foi desenvolvido em Angular 19 para auxiliar desenvolvedores a aprenderem e consultarem funções do ecossistema Protheus.

## Tecnologias

- **Angular 19** (Standalone Components)
- **CSS3** (Variáveis, Flexbox, Grid, Animações)
- **Glassmorphism Design**

## Funcionalidades

- **Dashboard**: Visão geral e atalhos rápidos.
- **Biblioteca de Funções**: Busca e filtro por categorias (Conversão, Banco de Dados, Arrays, etc).
- **Detalhes da Função**: Sintaxe, parâmetros, retorno e exemplos de código copiáveis.
- **Quiz Interativo**: Teste seus conhecimentos com feedback imediato.
- **AI Chat**: Tire dúvidas simuladas com um assistente virtual.

## Como Rodar

1. Certifique-se de ter o Node.js instalado (v18 ou superior).
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```
4. Acesse `http://localhost:4200` no seu navegador.

## Estrutura do Projeto

- `src/app/components`: Componentes da aplicação (Dashboard, Functions, Quiz, Chat).
- `src/app/services`: Serviços de dados (AdvplDataService).
- `src/styles.css`: Estilos globais e sistema de design.
